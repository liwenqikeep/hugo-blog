# Liwenqi's Blog

个人技术博客，记录开发学习过程中的知识点与踩坑记录。
- https://liwenqikeep.github.io/

## 内容方向

- **Java** - 核心技术栈、源码解析、实战经验
- **LLM / AI 应用** - 大语言模型实践、AI 工具使用

## 技术栈

- **框架**: Hugo
- **主题**: 自定义主题
- **部署**: GitHub Pages

## 快速开始

```bash
# 本地预览
hugo server -D

# 构建静态文件
hugo
```

## 目录结构

```
hugo-blog/
├── content/          # 文章内容
│   ├── post/         # 技术文章
│   └── page/         # 独立页面
├── layouts/          # 模板文件
├── static/           # 静态资源
├── assets/           # 资源文件
└── hugo.yaml         # 站点配置
```
